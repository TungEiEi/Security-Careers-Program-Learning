# üöÄ NextGen Security Careers Program - Cybersecurity Learning Journey

This repository contains all my personal notes, lecture materials (PDFs), code, and assignments from the 10-module **NextGen Security Careers Program** conducted by ARIT.

The purpose of this repository is to organize all course content for future review and to document my learning progress through each module.

The repository is structured into folders based on the 10 training modules:

* [**Module 01: Network Fundamentals**](./Week_01/)
    * *Topics:* Networking basics, TCP/IP Model, Subnetting, and network analysis tools.
    * [View Materials & Assignment](./Week_01/)

* [**Module 02: Cybersecurity Fundamentals**](./Week_02/)
    * *Topics:* Core security principles, CIA Triad, threat landscapes, and security policies.
    * [View Materials & Assignment](./Week_02/)

* [**Module 03: Security Operations (SOC)**](./Week_03/)
    * *Topics:* SOC operations, SIEM (Security Information and Event Management), and threat monitoring.
    * [View Materials & Assignment](./Week_03/)

* [**Module 04: Incident Response**](./Week_04/)
    * *Topics:* The Incident Response Lifecycle, handling security incidents, and procedures.
    * [View Materials & Assignment](./Week_04/)

* [**Module 05: Threat Intelligence**](./Week_05/)
    * *Topics:* Cyber Threat Intelligence (CTI), Indicators of Compromise (IOCs), and Threat Hunting.
    * [View Materials & Assignment](./Week_05/)

* [**Module 06: Digital Forensics**](./Week_06/)
    * *Topics:* Digital forensic investigation processes, evidence collection, and analysis.
    * [View Materials & Assignment](./Week_06/)

* [**Module 07: Penetration Testing (Part 1)**](./Week_07/)
    * *Topics:* Pentesting phases, reconnaissance, and scanning techniques.
    * [View Materials & Assignment](./Week_07/)

* [**Module 08: Penetration Testing (Part 2)**](./Week_08/)
    * *Topics:* Gaining access (exploitation), post-exploitation, and report writing.

    * *Topics:* Security on cloud platforms (AWS, Azure, GCP), security groups, and IAM.
