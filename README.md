This repository contains all my personal notes, lecture materials (PDFs), code, and assignments from the **SEC Career Program** conducted by ARIT.

## üìö Syllabus & Content

The repository is structured into folders based on the training modules:

* [**Module 01: Network Security**](./Week_01/)
    * *Topics:* CIA Triad, OSI & TCP/IP models, network architecture (DMZ, VLANs), Firewalls, IDS/IPS, Access Controls (RBAC, MAC/DAC), secure protocols (HTTPS, SSH, IPsec), and insecure protocols (FTP, Telnet).

* [**Module 02: Python Programming Basics**](./Week_02/)
    * *Topics:* Print, Data Types, Input, Operations, Strings, If Statements, Loops, Data Structures (List, Dictionary), Functions, Exceptions, and Object-Oriented Programming (Classes, Inheritance).

* [**Module 03: Cybersecurity Security Tools**](./Week_03/)
    * *Topics:* Overview of tools for Network Monitoring (Wireshark), Vulnerability Scanning (Nmap, Nessus), Web App Security (ZAP, Burp Suite, Nikto), Malware Analysis (VirusTotal), SIEM, and Penetration Testing (Metasploit, Hydra).

* [**Module 04: Core Security Principles**](./Week_04/)
    * *Topics:* Core concepts (CIA Triad, IAAA), Risk Management, Security Controls, IAM (MFA, Least Privilege), frameworks (ISO 27001, NIST), endpoint security (EDR/XDR), and security awareness (Phishing, Social Engineering).
